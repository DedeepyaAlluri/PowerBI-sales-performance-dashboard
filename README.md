# Sales Performance Dashboard – Power BI

![Dashboard Preview](Dashboard..png)

## Project Overview
This project demonstrates an end-to-end **Sales Performance Dashboard** built using **Power BI Desktop**. The dashboard provides actionable insights into sales trends, product performance, and revenue contributions across segments and categories.

The goal of this project is to **analyze sales data**, identify key performance drivers, and create an interactive dashboard that can assist decision-makers in optimizing business performance.



## Key Metrics / KPIs
- **Total Revenue** – Overall revenue across all products and segments  
- **YTD Revenue** – Year-to-date revenue trend  
- **Total Orders** – Total number of sales orders processed  
- **Total Quantity** – Quantity of products sold  
- **Average Order Value** – Revenue per order



## Visualizations
- **Monthly Revenue Trend** – Shows revenue changes month-over-month  
- **Sales Contribution by Segment** – Displays which customer segments contribute most to revenue  
- **Top Products by Revenue** – Highlights the highest revenue-generating products  
- **Category Contribution Analysis** – Shows revenue contribution by product category



## Key Insights
- Revenue peaked in **April**, driven by higher **average order value**  
- **Retail segment** contributes the majority of total revenue  
- **Accessories category** contributes approximately **92% of total revenue**  
- **Smartwatch** is the top revenue-generating product  

> These insights can help business leaders focus on high-performing products and segments for strategic growth.



## Tools & Skills
- **Power BI Desktop** – Dashboard creation and data visualization  
- **DAX** – Creating calculated measures for KPIs  
- **Data Modeling** – Building relationships between tables  
- **Power Query** – Data cleaning, transformation, and preparation  



## Dataset
- The dataset includes sales records with details like **Order Date, Product, Category, Segment, Revenue, Quantity**  
- Data transformations and cleaning were performed using **Power Query**



## How to Use
1. Download the `.pbix` file from this repository.  
2. Open the file in **Power BI Desktop**.  
3. Interact with slicers and filters to explore insights.
