## Sales Performance Dashboard – Power BI

This project showcases an end-to-end sales performance dashboard built using Power BI.

### KPIs
- Total Revenue
- YTD Revenue
- Total Orders
- Total Quantity
- Average Order Value

### Visuals
- Monthly Revenue Trend
- Sales Contribution by Segment
- Top Products by Revenue
- Category Contribution Analysis

###Dashboard..png

### Key Insights
- Revenue peaked in April driven by higher average order value.
- Retail segment contributes the majority of total revenue.
- Accessories category contributes ~92% of total revenue.
- Smartwatch is the top revenue-generating product.

### Tools & Skills
- Power BI Desktop
- DAX
- Data Modeling
- Data Transformation (Power Query)
