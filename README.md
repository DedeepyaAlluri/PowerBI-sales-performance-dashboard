# Sales Performance Dashboard – Power BI

![Dashboard Preview](dashboard.png)

## Project Overview
This project demonstrates an end-to-end **Sales Performance Dashboard** built using **Power BI Desktop**. 
The dashboard is designed to analyze sales trends, product performance, and revenue contributions across customer segments and categories. 
It provides actionable insights for business decision-making and helps identify key areas for business growth.

---

## Dataset
The dataset contains sales records with the following key columns:
- **Order Date**
- **Product**
- **Category**
- **Segment**
- **Revenue**
- **Quantity**

Data transformations and cleaning were performed using **Power Query**, including:
- Removing duplicates
- Handling missing values
- Creating calculated columns for analysis

---

## Key Metrics / KPIs
- **Total Revenue** – Overall sales revenue
- **YTD Revenue** – Year-to-date revenue trend
- **Total Orders** – Total number of orders
- **Total Quantity** – Quantity of products sold
- **Average Order Value** – Revenue per order

---

## Visualizations & Insights
- **Monthly Revenue Trend** – Shows revenue changes month-over-month  
- **Sales Contribution by Segment** – Displays which customer segments contribute most to revenue  
- **Top Products by Revenue** – Highlights highest revenue-generating products  
- **Category Contribution Analysis** – Shows revenue contribution by product category  

**Key Insights:**
- Revenue peaked in **April** due to higher average order value  
- **Retail segment** contributes the majority of total revenue  
- **Accessories category** contributes approximately 92% of total revenue  
- **Smartwatch** is the top revenue-generating product

## How to Use
1. Download the `.pbix` file from this repository.  
2. Open it in **Power BI Desktop**.  
3. Use slicers and filters to explore insights.  
4. Click visuals to cross-filter other charts for interactive analysis.
